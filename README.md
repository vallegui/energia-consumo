Este projeto foi desenvolvido como parte de uma missão de iniciação em tecnologia. O objetivo é fornecer uma ferramenta simples e eficaz para que usuários possam estimar o impacto de seus aparelhos na conta de luz.

## 🚀 Como funciona?
O sistema solicita dados básicos de uso e aplica a fórmula matemática padrão de consumo elétrico para gerar uma estimativa mensal.

### 📐 Fórmula Utilizada
O cálculo é baseado na seguinte equação:
$$Consumo_{kWh} = \frac{Potência(W) \times Horas_{Dia} \times 30}{1000}$$
